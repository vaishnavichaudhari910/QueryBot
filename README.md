# QueryBot 🤖✨


### AI-Powered Chat & Image Generation Platform

🔗 **Live Demo:**https://query-bot-tawny.vercel.app/
---

## 📌 Overview

**QueryBot** is a full-stack AI-powered web application that allows users to interact with an intelligent chatbot and generate images using advanced AI models. It provides a seamless conversational experience along with a premium subscription system.

---

## ✨ Features

* 💬 **AI Chat System**
  Real-time conversational AI powered by Gemini API

* 🎨 **Image Generation**
  Generate images from text prompts

* 🔐 **Authentication & Security**
  JWT-based authentication with protected routes

* 📜 **Conversation History**
  Stores and displays previous chats

* 🖼️ **Image Gallery**
  View and manage generated images

* 💳 **Premium Subscription**
  Credit-based system using Stripe integration

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* Tailwind CSS

**Backend:**

* Node.js
* Express.js

**Database:**

* MongoDB

**Authentication:**

* JWT (JSON Web Tokens)

**APIs & Services:**

* Gemini API (AI Chat & Image Generation)
* ImageKit (Image Handling)
* Stripe (Payments & Subscriptions)
