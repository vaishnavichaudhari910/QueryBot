# QueryBot 🤖✨

QueryBot is an AI-powered application similar to **ChatGPT**.  
It allows users to **chat with AI** and also **generate images** from prompts.  
Built with a modern full-stack architecture, it is designed to be fast, scalable, and secure.

---

## 🚀 Features
- 💬 Chat with AI (like ChatGPT)  
- 🖼️ Generate images from text prompts  
- 🔑 Secure authentication with Clerk  
- ⚡ Powered by Express.js backend & Vercel deployment  
- 🌐 Modern client with React + Vite  

---

## 🛠️ Tech Stack
- **Frontend:** React (Vite), Tailwind CSS  
- **Backend:** Express.js, Node.js  
- **Auth:** Clerk  
- **AI Services:** Gemini API (for text), Image API (for image generation)  
- **Payments:** Stripe (for premium credits)  
- **Database:**  PostgreSQL
